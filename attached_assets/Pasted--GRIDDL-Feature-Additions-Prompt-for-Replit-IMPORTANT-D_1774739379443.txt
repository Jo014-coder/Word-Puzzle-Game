**Files to modify:** `contexts/GameContext.tsx` only (add streak reward logic inside the existing daily win handling)

## 3. SOUND EFFECTS

**What to build:**
- Install `expo-av` for audio (it is already compatible with this Expo project)
- Add subtle sound effects for:
  - Peg placed → soft tick/click sound
  - Row submitted → whoosh or confirm sound
  - Correct guess (win) → short fanfare or chime
  - Wrong guess (lose) → low thud or buzz

- Source free sounds from `https://freesound.org` or use Expo's built-in audio capabilities with simple generated tones if assets are unavailable
- Add a **Sound toggle button** (🔊/🔇 icon) in the top-right area of `HomeScreen.tsx` header, saving the preference to `AsyncStorage` so it persists
- If sound is toggled off, skip all audio playback — never crash if audio fails to load

**Files to modify:** `HomeScreen.tsx` (sound toggle UI), `components/GameScreen.tsx` (trigger sounds on peg/row events), create `utils/sounds.ts` to centralise all audio logic

## 4. HOW TO PLAY — Onboarding + Persistent Help Button

**What to build:**

### First-launch tutorial:
- On first ever app launch (check `AsyncStorage` for a `'hasSeenTutorial'` key), show a `HowToPlayModal` before the home screen is interacted with
- The modal should explain:
  1. **Goal:** Crack the secret colour code
  2. **Pegs:** Tap a slot, then tap a colour from the bottom palette to place it
  3. **Feedback pins:** 🟢 Green = right colour, right position | 🟡 Yellow = right colour, wrong position | ⚫ Grey = colour not in code
  4. **Extreme mode:** One feedback pin is fake — don't trust them all!
- Use 3–4 simple slides/steps with a "Next" and "Got it!" button
- Match the existing dark theme, `Colors`, and `Inter` fonts exactly — no new colour values

### Persistent Help button:
- Add a small `?` icon button in the top area of `HomeScreen.tsx` (near the GRIDDL title or in a corner)
- Tapping it opens the same `HowToPlayModal` at any time
- The modal should have a close (✕) button so the player can dismiss it

**Files to modify:** `HomeScreen.tsx` (add `?` button), create `components/HowToPlayModal.tsx`
**Do NOT modify:** `GameScreen.tsx`, `ResultScreen.tsx`, `ShopScreen.tsx`, or any context/game logic files for this feature
